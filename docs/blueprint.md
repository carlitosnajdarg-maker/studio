# **App Name**: BarPool Menu

## Core Features:

- Menu Display: Display menu items (drinks, food, chips, cocktails) in a card-based layout.
- Image Gallery: Provide quick-access buttons to view images of drinks and food items.
- Tip Options: Include quick buttons to easily tip staff.
- Staff Ratings: Allow users to view staff ratings via quick-access buttons.

## Style Guidelines:

- Primary background: Deep plum/black (#120108) to highlight neon elements.
- Card background: Secondary color (#1A020C) for content cards.
- Primary accent color: Bright magenta (#FF008A) for headings and active elements.
- Secondary accent color: Electric cyan (#00F0FF) for labels and metadata.
- Headings: 'Space Grotesk' (sans-serif) for a rounded, extended futuristic feel, bold, sentence case, tight letter spacing. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) for standard text, regular/medium weight, 1.4 line height. Note: currently only Google Fonts are supported.
- Card layout: Vertical stack for content cards.
- Grid system: 2-column symmetrical grid for menu items.
- Use simple icons for quick actions like image view and tipping.
